template<typename Iterator>
class wordring::detail::const_list_trie_iterator< Iterator >

文字列の集合をTrie構築に適した形で走査するイテレータ

書籍「形態素解析の理論と実装」で紹介されている三つ組みによる走査の実装。

テンプレート引数
Iteratorベースとなる std::vector<std::string>::const_iterator のような文字列の集合に対するイテレータ
文字列を値とするSTLコンテナを指すイテレータを想定している。
  • #による空遷移も状態番号を持つ事に注意。この動作は、ダブルアレイ構築に必要なため設定した。
