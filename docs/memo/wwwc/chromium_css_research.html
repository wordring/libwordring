メンバとして実装されなかった生成規則

<selector-list> 規則のどこからも参照されていない。つまり、根となる予定の規則。
<simple-selector-list> 規則のどこからも参照されていない。つまり、根となる予定の規則。
<relative-selector-list> 規則のどこからも参照されていない。つまり、根となる予定の規則。
<relative-selector> 相対セレクタ。　引数として渡される scope: 要素があるため、セレクタ文字列が '>'、'+'、'~' から始まる。
<ns-prefix> <type-selector> 相当の関数内に直接実装される。関数に分離していない。
<wq-name> <type-selector> と <attribute-selector> 相当の関数内に直接実装される。関 数に分離していない。
<subclass-selector> <type-selector> 以外の simple セレクタ。
