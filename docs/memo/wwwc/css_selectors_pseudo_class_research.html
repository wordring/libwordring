疑似クラスの調査

疑似クラスのマッチにノード以外何が必要か調べるため、個別の疑似クラスを調査した。規格上は、単体セレクタの扱いで、ノードと単純に比較できるかのように 書かれている。

結論として、マッチに必ずノード以外の情報を必要とする疑似クラスは、 :scope のみだった。

https://developer.mozilla.org/ja/docs/Web/CSS/:scope

現在、スタイルシートで使用したときは、現時点ではスコープ付きの要素を明確に確立する方法がないため、 :scope は :root と同じです。 DOM API (querySelector(), querySelectorAll(), matches(), Element.closest() など) で使用したときは、 :scope はメソッドを呼び出した要素を選択します。
